# Scholarship Finder

A Python command-line tool that helps students discover 
scholarships they are eligible for based on their academic profile.

## Inspiration
Built because I personally struggled finding the right scholarship 
opportunities as an Indian student. This tool is for every student 
who doesn't know what opportunities exist for them.

## Features
- Find scholarships based on your percentage and preferred country
- View all available scholarships in the database
- Save and view your search history
- Currently includes scholarships for South Korea, Japan and Germany

## Scholarships in Database
- GKS-U (Global Korea Scholarship)
- MEXT Scholarship (Japan)
- Underwood International Scholarship (Yonsei)
- DAAD Scholarship (Germany)

## How to Run
1. Make sure Python 3 is installed
2. Clone this repository
3. Run:

    python scholarshipfinder.py

## Technologies Used
- Python 3
- File Handling
- Functions and Modular Programming

## About
This project was built to solve a real problem faced by 
thousands of Indian students who are unaware of fully 
funded international scholarship opportunities.
